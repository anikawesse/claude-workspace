## Frage: Wie viel Adspend täglich für €300 Bruttoumsatz?

### Rechnung über ROAS

```
Ziel Bruttoumsatz:  €300 / Tag
Ø ROAS (Mai):        3,22
Benötigter Adspend:  300 ÷ 3,22 = ~€93 / Tag
```

**Antwort: ca. €90–95 / Tag** – das ist etwa 3,3× dein aktueller Adspend.

## Breakeven

```
Verdienst-Quote: 74 %
Breakeven-ROAS: 1 ÷ 0,74 = 1,35
```

Solange dein ROAS über **1,35** liegt, machst du Gewinn – egal wie hoch der Adspend ist.
Unter 1,35 = Verlust. Derzeit liegst du im Schnitt weit drüber (3,22).

## Frage: Wie viel Adspend täglich für €300 Gewinn?

### Rechnung

```
Gewinn      = (Adspend × ROAS × 0,74) − Adspend
€300        = Adspend × (3,22 × 0,74 − 1)
€300        = Adspend × 1,383
Adspend     = 300 ÷ 1,383 = ~€217 / Tag
Bruttoumsatz = 217 × 3,22 = ~€699 / Tag
```

**Antwort: ca. €217 / Tag Adspend** für €300 Gewinn täglich.
