## Wiederkehrende Bausteine (über alle Skripte)

- **Kernframework:** Eisberg-Modell → **Auslöser** (Spitze, z. B. Futter/staubiges Heu) vs. **Ursache** (großer Brocken darunter = emotionales Thema).
- **Fragetechnik (Futter/Behandlung):** Verträgst du es? → Brauchst du es? → Priorisierung 1–10 (nichts unter 7 bestellen).
- **Aktive Codewörter:** `Tiergespräch` (bezahlte Einzel-Analyse, Infovideo per Messenger — evergreen), `Kopfkino` (Mini-Kurs gegen Angst).
- **Engagement-Codewörter in Lives:** `interessant` / `ich`.
