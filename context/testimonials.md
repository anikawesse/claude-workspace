## Übergreifende Muster (Marketing-Gold)

| Thema | Was sie überrascht / wertvollstes Learning |
|---|---|
| TTP | Mentale Arbeit wirkt stärker als erwartet |
| BLP | Struktur & roter Faden — endlich wissen wo anfangen |
| Futter-Fahrplan | "Nicht so kompliziert wie gedacht" — AHA-Momente |
| Offenstallplaner | "Hätte ich das früher gewusst" — Fehler vermeiden |
| Alle | Selbstständig werden ohne Trainer vor Ort |

**Häufigste Kaufmotive:**
- Endlich einen roten Faden / Struktur haben
- Selbstständig ohne Trainer vor Ort trainieren können
- Sicherheit gewinnen — bei sich UND beim Pferd
- Verstehen (nicht nur machen) — "warum" begreifen
